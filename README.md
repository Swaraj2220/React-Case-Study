ProcureCraft Slate – Smart Procurement Dashboard

A modern React.js-based procurement management dashboard designed to streamline procurement operations, inventory management, vendor verification, cost analysis, and logistics planning. The application provides a centralized interface for managing purchase requests, tracking price changes, verifying vendors, monitoring inventory rules, optimizing supply routes, and balancing warehouse stock.

Project Description

ProcureCraft Slate is a procurement and inventory management dashboard built using React JS. It helps organizations manage purchasing workflows efficiently by providing tools for order tracking, vendor verification, budget monitoring, inventory management, and logistics optimization.

The dashboard is designed with a clean and responsive user interface, allowing users to quickly access procurement-related information and make data-driven decisions.

Installation & Setup

Prerequisites

Make sure you have installed:

* Node.js (v18 or above)
* npm

Verify installation:

node -v
npm -v

Clone Repository
git clone https://github.com/Swaraj2220/React-Case-Study.git

Move into the project folder:
cd React-Case-Study

Install Dependencies
npm install

Run Development Server
npm run dev

The application will start on:
http://localhost:5173

Build for Production
npm run build

Preview Production Build
npm run previewLearning Outcomes
